Case Study

Ana Martini
Course Platform

A full-stack learning management system built for a professional makeup artist — from registration to course delivery, all under one roof.

View live site

Stack

  • Backend Django 6 · Django REST Framework · Simple JWT
  • Frontend React 19 · Vite · Zustand · React Query · Axios
  • Database PostgreSQL (Railway)
  • Auth JWT with token rotation and blacklisting
  • Email Gmail SMTP — password resets & admin notifications
  • Static files Whitenoise
  • Hosting Railway — Django API + React static site + PostgreSQL
  • Video YouTube embed — no storage costs, scales freely

About the project

Ana Martini is a professional makeup artist with 15 years of experience across social, artistic, children's, and carnival makeup. She needed a platform to sell and deliver her online courses — simple for her to manage, clean for her students to use.

The result is a full-stack LMS with course browsing, user registration, JWT-authenticated access control, and a built-in messaging system where students can contact the instructor directly and request course access.

Course content is structured as Courses → Modules → Videos, with a mix of free preview content and locked paid lessons. Students can see what's inside before committing.

Key decisions

YouTube for video hosting. Rather than storing video files in the cloud — which adds storage and bandwidth costs immediately — all course videos are embedded from YouTube. This keeps hosting costs at zero while the business grows, and can be migrated to a dedicated CDN later when scale justifies it.

Manual access grant. There's no payment gateway integrated yet. Students request access through the messaging system, Ana confirms payment externally (bank transfer, etc.), and grants access manually via the admin panel. Simple and practical for an early-stage operation.

Flat admin. No Django admin — a custom React admin panel gives Ana full control over users, courses, modules, videos, and messages without any technical knowledge required.
