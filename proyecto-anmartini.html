Plataforma de Cursos

Un sistema de gestión de aprendizaje full-stack construido para una maquilladora profesional - desde el registro hasta la entrega de cursos, todo en un solo lugar.

Ver sitio en vivo

Stack

  • Backend Django 6 · Django REST Framework · Simple JWT
  • Frontend React 19 · Vite · Zustand · React Query · Axios
  • Base de datos PostgreSQL (Railway)
  • Auth JWT con rotación y blacklisting de tokens
  • Email Gmail SMTP - recuperación de contraseña y notificaciones
  • Archivos estáticos Whitenoise
  • Hosting Railway - API Django + sitio React + PostgreSQL
  • Video Embed de YouTube - sin costos de almacenamiento

Sobre el proyecto

Annie Makeup es una maquilladora profesional con 15 años de experiencia en maquillaje social, artístico, infantil y de murga. Necesitaba una plataforma para vender y entregar sus cursos online - simple de gestionar para ella, clara y fácil de usar para sus alumnas.

El resultado es un LMS full-stack con navegación de cursos, registro de usuarios, control de acceso autenticado con JWT, y un sistema de mensajería integrado donde los alumnos pueden contactar a la instructora y solicitar acceso a los cursos directamente.

El contenido está organizado en Cursos → Módulos → Videos, con una combinación de contenido gratuito de vista previa y lecciones de pago bloqueadas. Los alumnos pueden ver qué hay adentro antes de comprometerse.

Decisiones clave

YouTube para el hosting de videos. En lugar de almacenar archivos de video en la nube - lo que genera costos de almacenamiento y ancho de banda de inmediato - todos los videos de los cursos se embeben desde YouTube. Esto mantiene los costos de hosting en cero mientras el negocio crece, y puede migrarse a un CDN dedicado cuando la escala lo justifique.

Acceso concedido manualmente. No hay pasarela de pago integrada aún. Los alumnos solicitan acceso a través del sistema de mensajería, Annie confirma el pago externamente (transferencia bancaria, etc.), y otorga el acceso manualmente desde el panel de administración. Simple y práctico para una operación en etapa inicial.

Panel admin propio. Sin el admin de Django - un panel de administración personalizado en React le da a Annie control total sobre usuarios, cursos, módulos, videos y mensajes sin necesitar conocimientos técnicos.
